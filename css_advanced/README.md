# CSS Advanced Project - ALX HTML & CSS

## Project Description
This project focuses on creating a modern and responsive webpage using **HTML and CSS**. The main goal is to practice advanced CSS styling techniques, including typography, layout, colors, and responsive design.

The design for this project was provided in **Figma**. All assets such as logos and images were downloaded and implemented in the webpage. 

## Project Objectives
- Convert Figma design into HTML and CSS.  
- Practice advanced CSS styling: colors, borders, shadows, fonts, and hover effects.  
- Ensure a **responsive layout** that works on desktop, tablet, and mobile.  
- Add a **logo** to the header.  
- Customize the `<article>` section fully with unique styling.  

## Tools & Resources
- HTML5, CSS3  
- Fonts: [Source Sans Pro](https://fonts.google.com/specimen/Source+Sans+Pro) and Spin-Cycle-OT
